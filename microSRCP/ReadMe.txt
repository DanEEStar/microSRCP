Sourcen fuer Arduino IDE und Beispieldateien RocRail.

Arduino IDE
-----------
Dateien aus dem microSRCP/arduino Verzeichnis in Arduino Sketchbook-
Verzeichnis kopieren (siehe File -> Preferences in der 
Arduino IDE).

Mittels File -> Sketchbook koennen dann die einzelne
Board Sketches geoeffnet, compiliert und ins Board
geladen werden.

RocRail
-------
Die SRCP Zentrale kann einfach mittels dem User-Interface von 
RocRail erstellt werden.

Dazu ist eine srcp Zentrale einzutragen und je nach Sketch
die Sub Bibliothek TCPIP oder RS232 einzutragen.

In den Verzeichnissen RocRail befinden sich Beispiele der
rocrail.ini (Konfiguration Zentrale) und dazupassende
plan.xml Dateien.
